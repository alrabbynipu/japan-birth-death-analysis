# Japan Birth & Death Ratio Analysis (1947–2022)

## Overview
Exploratory Data Analysis (EDA) of Japan's demographic trends over 75 years
using birth, death, fertility and population data.
Conducted as an academic project using Python.

## Dataset
- Source: japan_birth.csv
- 77 rows × 53 columns
- Coverage: 1947–2022

## Analyses Conducted

| 5 | Male vs Female Birth Trend |
| 6 | Scatter Plot: Mother Age vs Total Births |
| 7 | Birth Order Distribution (2020) |
| 8 | Total Fertility Rate Over Time |
| 9 | Life Expectancy at Birth |

## Key Findings
- Deaths surpassed births around 2005, marking the start of natural population decline
- Total fertility rate has fallen well below the replacement level of 2.1
- Average mother age at first birth has steadily increased since 2000
- Life expectancy has consistently risen despite declining birth rates
